The Raspberry Pi

See: http://www.raspberrypi.org/ebens-keynote-at-pycon/

The issue at hand...

  • There was a time when, pursuing the interesting things one wanted to do, one had to choose NOT to program a PC.
  • This created a generation of child programmers who showed up at the doorstep of Cambridge already with advanced knowledge.
  • So much so that their engineering programs had to spend weeks teaching them what they did not know they did not know.
  • In the time since, as PCs and consoles evolved, the impetus to learn to program went away.
  • And now Cambridge is having trouble finding enough students, and has to spend extra time catching them up to what the courses used to be able to take for granted.

And lo, Raspberry Pi was born

with specific goals in mind

  • If a new generation of coders was to be raised, the desire to code starting young had to be reintroduced.
  • This interface had to be programmabe
  • This interface had to be interesting
  • This interface had to be robust
  • This interface had to be affordable

A great success

  • 100,000 orders were placed on their first day... exhausting their original production estimates within 15 minutes.
  • Thankfully, Premier Farnell and RS Electronics, recogizing the same decline in engineering students, stepped up and now do the heavy lifting production under license.
  • The things folk have been using the Raspberry pi for  has gone far beyond anything the foundation had imagined.
  • At PyCon 2013, days of raspberry pi centric workshops brought the device and kids with little to no experience together, and many new enthusiasts were born.
